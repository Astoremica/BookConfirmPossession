# 簡易仕様書

### 作者
YoNa
### アプリ名
マンガ購入確認アプリ

#### コンセプト
過去に買ったマンガかどうか、その場で知ることができる。

#### こだわったポイント
使用するのは、アプリと、本のバーコードのみ。
大型スマホでの使用も考え、主要機能を画面下部に集める。

## 開発環境
### 開発環境
Xcode 12.4

### 開発言語
Swift 5.1

## 動作対象端末・OS
### 動作対象OS
iOS 14.2(制作開始)
14.5.1(最終動作確認)

## アプリケーション機能

### 機能一覧

- 履歴編集：過去の購入履歴を削除できる。
- バーコードスキャン：本にあるISBNコードを読み取る
- 書籍情報表示：ISBNコードをもとに、APIを用いて情報を取得、購入済みか未購入を表示


### 画面一覧

- 履歴：過去に購入したマンガを確認できる。
- 履歴詳細：本のタイトル、表紙、購入日の確認ができる。
- 書籍情報表示：購入済みか未購入を表示、表紙、タイトル、購入済みなら購入日・未購入であれば発売日
- スキャンリスト表示：一度に複数のマンガを購入する時に読み取ったマンガの表紙一覧を表示する


### 使用しているAPI,SDK,ライブラリなど
- 楽天ブックスAPI
- Realm（CocoaPods）
